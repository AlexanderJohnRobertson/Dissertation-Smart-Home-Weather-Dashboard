Docker Commands

Build: docker build -t alexjrobertson/smarthouseweatherdashboard:latest .

Run: docker container run -d -p 5000:5000 alexjrobertson/smarthouseweatherdashboard:latest

Push: docker push alexjrobertson/smarthouseweatherdashboard:latest

Pull: docker pull alexjrobertson/smarthouseweatherdashboard
